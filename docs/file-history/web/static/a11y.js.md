**JavaScript** · 134 lines · 5,493 bytes · 1 commit(s) · first 2026-06-07 · last 2026-06-07

## Purpose

Accessibility & theme controls for the World Downloader web console. Builds a floating menu, persists choices to localStorage, and applies them as data-* attributes on <html> (see the matching rules in style.css). Includes presets for common needs (ADHD focus, autism-calm, dyslexia/low-vision reading) plus light/dark/high-contrast themes.
